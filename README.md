# Portfolio

Personal portfolio showcasing ML/AI projects with an AI-powered navigation assistant.

## Features

- **Project Showcase**: Interactive cards highlighting ML/AI work
- **AI Guide**: Source-grounded assistant for navigating projects
- **Warm Terminal Design**: Cozy IDE aesthetic with orange accents

## Tech Stack

- Next.js + TypeScript
- Tailwind CSS
- Vercel Edge Functions
- OpenAI Embeddings + Groq LLM

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Projects

1. **Pickleball CV** - Computer vision for sports analytics
2. **Spotify RAG** - Multimodal retrieval + DJ Assistant
3. **AIM VIP** - NPU hardware acceleration research
4. **CV Coursework** - Image classifiers + SAM 2 research
5. **HuggingFace Clone** - CI/CD, AWS deployment
