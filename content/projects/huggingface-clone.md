# HuggingFace Clone

## Demo

Video demo coming soon - will show model upload and search functionality.

## Overview

A simplified clone of the HuggingFace Hub built for a Purdue software engineering course. Supports model uploads, metric evaluation, search, and deletion. Full-stack application deployed on AWS.

## How It Works

### REST API
Designed and implemented Flask/OpenAPI endpoints for:
- Model metric evaluation
- Search functionality
- Upload and deletion
- User authentication

### Database
SQLite for development testing. Designed schema for model metadata, user accounts, and evaluation metrics.

### Deployment
- Docker containerization for consistent environments
- AWS deployment (EC2, S3, CloudWatch)
- CI/CD pipeline for automated testing and deployment

## Tech Stack

- **Backend:** Python, Flask, OpenAPI
- **Frontend:** React
- **Database:** SQLite
- **Infrastructure:** Docker, AWS (EC2, S3, CloudWatch)
- **DevOps:** CI/CD pipeline, Git
